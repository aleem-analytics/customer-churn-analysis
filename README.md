# 📊 Customer Retention & Churn Analysis Dashboard

## 🧠 Overview  
This project analyzes customer churn behavior to identify patterns in retention, revenue contribution, and customer segments. The goal is to provide insights that help businesses reduce churn and improve customer lifetime value.

---

## 🛠 Tools Used  
- Power BI  
- DAX (Data Analysis Expressions)  
- CSV Dataset  

---

## 📁 Dataset  
The dataset contains customer information including:
- CustomerID  
- Age  
- Gender  
- Subscription Type  
- Monthly Spend  
- Tenure  
- Churn (Yes/No)  

---

## 🧹 Data Preparation  
- Loaded and structured customer dataset in Power BI  
- Created calculated measures:
  - Total Customers  
  - Churned Customers  
  - Churn Rate  
- Applied filters to analyze churn-specific behavior  
- Aggregated data by subscription type, tenure, and gender  

---

## 📊 Dashboard Features  
- Total Customers, Churned Customers, and Churn Rate KPIs  
- Churn analysis by subscription type  
- Average monthly spend by customer segment  
- Churn distribution across tenure  
- Customer distribution by gender  

---

## 🔍 Key Insights  
- Customers on the Basic subscription plan show the highest churn levels  
- Customers with shorter tenure are more likely to churn  
- Premium customers generate the highest average revenue and are more stable  
- Churn is evenly distributed across gender, indicating gender is not a major factor  

---

## 📌 Conclusion  
This analysis highlights key areas where businesses can focus to improve customer retention, especially by enhancing early customer experience and optimizing lower-tier subscription plans.
